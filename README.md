# SentinX Hybrid IDS with AI‑Driven Dashboard

## 📘 Project Overview
SentinX Hybrid IDS is a cybersecurity project that combines **signature‑based detection (Suricata)** and **anomaly detection (Zeek)** to provide a robust intrusion detection system. It integrates with **Filebeat, Elasticsearch, and Kibana/Power BI dashboards** for real‑time monitoring and visualization.

---

## 🎯 Objectives
- Detect and analyze suspicious network traffic using a hybrid approach.
- Provide **real‑time alerts** with improved accuracy.
- Visualize intrusion data through **AI‑driven dashboards**.
- Build an examiner‑friendly academic project with practical implementation.

---

## ⚙️ System Architecture
- **Suricata** → Signature‑based IDS  
- **Zeek** → Behavioral/anomaly detection  
- **Filebeat** → Log collection  
- **Elasticsearch** → Log indexing and storage  
- **Kibana / Power BI** → Interactive dashboards
